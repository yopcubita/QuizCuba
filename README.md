# 🇨🇺 Quiz Cuba

App de trivia cubana hecha con React Native y Expo.

## Categorías
- 📜 Historia de Cuba
- 🗺️ Geografía cubana
- 🎭 Cultura y tradiciones
- 🎶 Música cubana
- 🥊 Deportes en Cuba
- ⭐ Personalidades cubanas

## Características
- 100 preguntas
- Temporizador por pregunta
- Ranking con detalle de respuestas
- Compartir resultados por WhatsApp
- Sonidos y efectos
- Funciona 100% offline

## Tecnologías
- React Native
- Expo SDK 51
- AsyncStorage
- React Navigation
